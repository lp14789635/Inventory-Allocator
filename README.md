# Inventory-Allocator
## To run the test program:
### Mac/Linux: 
In terminal, go to the src directory
run with 
```
./run.sh
```
If it does not work, run 
```
chmod +x ./run.sh
``` 
first and then "./run.sh"
### In Windows: 1. In terminal, go to the src directory
###             2. run "javac testClass.java"
###             3. run "java testClass"
