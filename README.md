# Inventory-Allocator
## To run the test program:
### Mac/Linux: 
In terminal, go to the src directory
run with 
```
./run.sh
```
If it does not work, run 
```
chmod +x ./run.sh
``` 
first and then 
```
./run.sh
```
### In Windows: 
In terminal, go to the src directory and compile with 
```
javac testClass.java
```
Then, run with 
```
java testClass
```
